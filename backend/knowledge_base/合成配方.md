# 饥荒联机版 合成配方
## 工具类
- 斧头：树枝×2 + 燧石×1
- 镐子：树枝×2 + 燧石×2
- 铲子：树枝×2 + 燧石×2 + 金子×1
- 锤子：树枝×3 + 燧石×3 + 金子×2

## 生存类
- 背包：树枝×4 + 草×4
- 保温石：金子×3 + 石头×3
- 帐篷：树枝×6 + 蜘蛛丝×4 + 绳子×3
- 蓑衣：草×6 + 树枝×2
- 雨伞：树枝×4 + 蜘蛛丝×3 + 猪皮×1

## 武器类
- 长矛：树枝×4 + 燧石×2
- 火腿棒：树枝×2 + 大肉×2 + 猪皮×1
- 狼牙棒：树枝×2 + 金子×2 + 猪皮×2
- 护甲：木甲=木头×8 + 绳子×2；大理石甲=大理石×6 + 绳子×2

## 科技类
- 科学机器：金子×4 + 木头×4 + 石头×4
- 炼金引擎：科学机器+金子×6 + 木头×4 + 石头×4 + 电路板×1
- 避雷针：金子×3 + 石头×3
- 灭火器：冰块×2 + 齿轮×2 + 电子元件×2
