# 饥荒联机版 实用食谱
## 肉丸
- 配方：肉度≥1 + 任意填充物
- 效果：回血+饱食，性价比最高
- 推荐：前期必备

## 肉汤
- 配方：大肉×2 + 怪物肉×1 + 填充物
- 效果：高饱食、高回血
- 适合：战斗、探图

## 火龙果派
- 配方：火龙果×1 + 填充物×3
- 效果：超高饱食、回血
- 适合：种田玩家

## 蛙腿三明治
- 配方：蛙腿×1 + 蔬菜×1 + 填充物×2
- 效果：性价比高，容易制作
- 适合：青蛙池塘玩家

## 华夫饼
- 配方：黄油×1 + 浆果×1 + 树枝×2
- 效果：超高回血
- 适合：BOSS战、应急回血

## 什锦菜
- 配方：任意三种蔬菜+填充物
- 效果：高精神回复
- 适合：精神不足时
