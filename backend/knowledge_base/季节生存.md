# 饥荒联机版 季节生存指南
## 秋季（开局季节）
- 特点：气候温和，无特殊危险
- 目标：建基地、科技、食物储备
- 注意：收集基础资源，准备过冬

## 冬季
- 特点：温度极低，会冻伤
- 必须：保暖石、冬帽、营火
- 危险：巨鹿、冰冻、食物短缺
- 技巧：多做炖锅，囤大肉，提前晒肉干

## 春季
- 特点：持续下雨，潮湿值
- 必须：雨伞、雨衣、隔热包
- 危险：青蛙雨、鹿鹅
- 技巧：基地垫高，避免积水

## 夏季
- 特点：高温自燃，植物枯萎
- 必须：灭火器、冰火、西瓜帽
- 危险：自燃、龙蝇
- 技巧：基地全覆盖灭火器，少用木质建筑
